# 📁 ByteReports — Documentos do Projeto

Este repositório centraliza toda a documentação oficial do projeto **ByteReports**, incluindo artefatos de gerenciamento, arquitetura, modelagem, testes e relatórios de sprint.



## 📄 Documentos

### 📘 Documentação Completa
Documento principal do projeto contendo visão geral, requisitos, arquitetura, decisões técnicas e demais informações consolidadas sobre o ByteReports.

### 🗂️ Estrutura Analítica do Projeto (EAP)
Representação visual da decomposição do trabalho do projeto em pacotes menores e gerenciáveis, facilitando o planejamento e o acompanhamento das entregas.

### 🧩 Model Canvas
Mapa visual do modelo de negócio do ByteReports, cobrindo proposta de valor, segmentos de clientes, canais, fontes de receita e estrutura de custos.

### 🗃️ Modelo Entidade-Relacionamento (MER)
Diagrama que representa a estrutura do banco de dados do sistema, descrevendo as entidades, atributos e os relacionamentos entre eles.

### 🎯 Pitch Deck
Apresentação desenvolvida para comunicar a proposta de valor do ByteReports a investidores e stakeholders, destacando o problema, a solução, o mercado e o diferencial competitivo.

### ✅ Plano de Testes
Documento que define a estratégia de qualidade do projeto, incluindo escopo dos testes, casos de teste, critérios de aceitação e responsabilidades da equipe de QA.

### 📊 Relatório: Sistemas de Diagnóstico de Hardware vs ByteReports
Análise comparativa entre soluções existentes de diagnóstico de hardware e a proposta do ByteReports, evidenciando os diferenciais e vantagens do sistema desenvolvido.

### 🔍 Resultados de Pesquisa
Compilação dos dados obtidos em pesquisas realizadas com usuárioso, utilizados para embasar decisões do produto.

### 📋 Relatórios de Sprint
Pasta contendo os relatórios de acompanhamento de cada sprint do projeto, registrando o progresso, entregas, impedimentos e retrospectivas da equipe.

### 🎥 Vídeo e Prints de demonstração do APP
Link para acesso do vídeo de demonstração do APP em DEMO V1.0: https://youtu.be/PW_X8Sy2Ars

### 🏁 Termo de Encerramento de Projeto
Documento formal que atesta a conclusão do projeto ByteReports, com as entregas realizadas, lições aprendidas e aceite das partes envolvidas.
